# Система мониторинга графиков

## Описание
- Веб-приложение для отображения графиков оборотов и температуры в реальном времени.

## Установка
1. Разместите все файлы на веб-сервере с поддержкой PHP
2. Убедитесь, что PHP настроен корректно
3. Откройте `index.html` в браузере

## Файлы
- `index.html` - главная страница приложения
- `server.php` - серверная часть для генерации данных
- `.htaccess` - конфигурация для Apache

## Функциональность
- Отображение графиков в реальном времени
- 5 различных цветовых схем графиков
- Автоматическое масштабирование осей
- Индикатор статуса соединения
- Адаптивный дизайн

## API
 - GET запрос к `/server.php` возвращает данные в формате: обороты,температура
